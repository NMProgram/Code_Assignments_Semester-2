# Overview
This exercise required the programmer to make use of a `switch (x)` statement to keep track of a user's position, using user input.
Each option would increment or decrement the x/y position and print the position afterwards.
I'm kinda confused why they didn't throw in the requirement to make this a loop, but I digress. 

# Code
```cs
static class Program
{
    static void Main()
    {
        int xPosition = 0;
        int yPosition = 0;
        Console.WriteLine("What direction (up/down/left/right) would you like to go?");
        string answer = Console.ReadLine() ?? "None";
        // Your code with the switch here :)
        switch (answer.ToLower())
        {
            case "up":
                yPosition++;
                break;
            case "down":
                yPosition--;
                break;
            case "right":
                xPosition++;
                break;
            case "left":
                xPosition--;
                break;
            default:
                Console.WriteLine("Invalid direction");
                break;
        }
        //
        Console.WriteLine($"Current position\nX:{xPosition}, Y:{yPosition}");
    }
}
```

# Overview
This exercise required the programmer to ask for a value for degrees `Celsius` as a `double`
and to convert that to a `state` of water.
Solid is less than or equal to `0 C`, liquid between `0 C` and `100 C` and gas beyond `100 C`.
I went for a completely new approach with this one, using `double.TryParse(string, out double value)` and splitting my `Program` class into two separate methods,
one being `Main()` and the other being the `if-statement`, just to make it a bit more organized.

# Code
```cs
static class Program
{
    static void IfStatement(double celsius)
    {
        if (celsius <= 0)
        {
            Console.WriteLine($"At {celsius} degrees Celsius, the water will be solid");
        }
        else if (celsius >= 100)
        {
            Console.WriteLine($"At {celsius} degrees Celsius, the water will be gas");
        }
        else
        {
            Console.WriteLine($"At {celsius} degrees Celsius, the water will be liquid");
        }
    }
    static void Main()
    {
        Console.WriteLine("What is the temperature of the water? (Celsius)");
        bool celsius = double.TryParse(Console.ReadLine(), out double value);
        if (celsius)
        {
            IfStatement(value);
        }
    }
}
```

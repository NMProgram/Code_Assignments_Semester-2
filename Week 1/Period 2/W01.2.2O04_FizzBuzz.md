# Overview
This exercise requires the programmer to ask a start and end number from the user to form a range of numbers to print.
Each number that is divisible by 3 prints "Fizz", divisible by 5 prints "Buzz" 
and if both requirements are met, "FizzBuzz" gets printed instead of the number.

I wanted to see if I could make this shorter, but I think a `switch` statement was the right move here, 
considering they're more efficient than `if-else if-else` statements.

# Code
```cs
static class Program
{
    static void Switch(int i)
    {
        switch (i % 3 == 0, i % 5 == 0)
            {
                case (false, false):
                    Console.WriteLine(i);
                    break;
                case (true, false):
                    Console.WriteLine("Fizz");
                    break;
                case (false, true):
                    Console.WriteLine("Buzz");
                    break;
                case (true, true):
                    Console.WriteLine("FizzBuzz");
                    break;
            }
    }
    static void Main()
    {
        Console.WriteLine("Start Number:");
        int start = int.Parse(Console.ReadLine()!);
        Console.WriteLine("End Number:");
        int end = int.Parse(Console.ReadLine()!);
        for (int i = start; i != end; i++)
        {
            Switch(i);
        }
    }
}
```
